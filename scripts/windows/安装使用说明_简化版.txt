===============================================
PrintProxy 安装器 - 简化版（推荐）
===============================================

版本：1.0.0
类型：开机自启动版本（无需密码）

主要特点：
-----------
✅ 右键"以管理员身份运行"即可完成安装
✅ 无需输入密码
✅ 无需配置 Windows 服务
✅ 自动开机启动
✅ 完整的打印机访问权限
✅ 程序以当前用户身份运行

安装步骤：
-----------
1. 右键点击 PrintProxySetup_1.0.0.exe
2. 选择"以管理员身份运行"
3. 点击 "Next" → 选择安装目录 → 点击 "Install"
4. 安装完成后，选择是否立即启动程序
5. 完成！

安装后：
-----------
- 程序安装在：C:\Program Files\PrintProxy\
- 程序会在您登录时自动启动
- 无需任何密码或额外配置
- 服务地址：http://0.0.0.0:8000

使用方法：
-----------
1. 程序安装后会自动启动
2. 每次电脑开机登录后会自动运行
3. 打开浏览器访问：http://localhost:8000/docs
4. 所有打印功能正常可用

手动控制：
-----------
启动程序：
  - 双击桌面快捷方式"Print Proxy"
  - 或从开始菜单 → PrintProxy → Print Proxy

停止程序：
  - 开始菜单 → PrintProxy → Stop Print Proxy
  - 或任务管理器结束 PrintProxy.exe 进程

卸载：
-----------
方式 1：运行安装器，选择卸载
方式 2：控制面板 → 程序和功能 → 卸载 Print Proxy
方式 3：运行 C:\Program Files\PrintProxy\Uninstall.exe

优势对比：
-----------
✅ 简化版（当前）：
   - 无需密码
   - 安装简单
   - 完整打印机访问
   - 以用户身份运行
   - 推荐使用

❌ 服务版（旧版）：
   - 需要输入密码

   - 不推荐

常见问题：
-----------
Q: 为什么需要管理员权限安装？
A: 需要写入 Program Files 目录和注册表启动项，仅安装时需要，运行时无需管理员权限。

Q: 程序会一直运行吗？
A: 是的，程序会在后台运行。如需停止，使用"Stop Print Proxy"快捷方式。

Q: 如何确认程序是否在运行？
A: 
   - 浏览器访问 http://localhost:8000/docs
   - 或任务管理器查看 PrintProxy.exe 进程

Q: 开机自启动能关闭吗？
A: 可以，通过任务管理器 → 启动 → 禁用 Print Proxy

Q: 会影响打印机使用吗？
A: 不会，程序仅提供 API 接口，不影响正常打印。

技术说明：
-----------
- 实现方式：注册表自启动（HKLM\...\Run）
- 运行方式：普通应用程序（非服务）
- 运行账户：当前登录用户
- 权限要求：普通用户权限（运行时）
- 端口：8000

文件说明：
-----------
主程序：PrintProxy.exe（单文件，包含所有依赖）
日志目录：C:\Program Files\PrintProxy\logs\
配置方式：环境变量
  - PRINT_PROXY_HOST (默认: 0.0.0.0)
  - PRINT_PROXY_PORT (默认: 8000)

对比旧版本：
-----------
简化版安装器：PrintProxySetup_1.0.0.exe
服务版安装器：PrintProxySetup_1.0.0_service.exe.bak

推荐使用简化版！
