# 项目结构说明

## 目录结构

```
print_proxy/
├── app/                    # 应用程序核心代码
├── docs/                   # 项目文档
│   ├── API文档.md
│   ├── CHANGELOG_配置系统.md
│   ├── README_安装使用.md
│   ├── 快速测试指南.md
│   ├── 配置快速参考.md
│   ├── 配置迁移指南.md
│   ├── 配置说明.md
│   ├── 配置系统实现总结.md
│   ├── 自启动权限问题修复方案.md
│   └── 需求文档.txt
├── scripts/                # 脚本工具
│   ├── docker/            # Docker 相关脚本
│   ├── windows/           # Windows 平台脚本
│   │   ├── start_server.ps1      # 服务启动脚本
│   │   ├── verify_registry.ps1   # 注册表验证脚本
│   │   └── ...
│   ├── show_config.py     # 配置显示工具
│   └── validate_config.py # 配置验证工具
├── tests/                 # 测试文件
├── .env.example          # 环境变量配置模板
├── config.yaml.example   # YAML 配置模板
├── README.md             # 项目主说明文档
└── requirements.txt      # Python 依赖列表
```

## 主要改动

### 文档整理
- 将根目录下的所有 `.md` 文档文件移动到 `docs/` 目录
- 保持文档内容不变，仅调整存放位置

### 脚本整理
- 将 `start_server.ps1` 移动到 `scripts/windows/`
- 将 `verify_registry.ps1` 移动到 `scripts/windows/`
- 保持脚本功能不变

### 路径引用更新
- 更新了所有文档中对移动文件的路径引用
- 更新了脚本中的路径引用
- 确保程序运行不受影响

## 使用说明

### 启动服务
```powershell
# 从项目根目录运行
.\scripts\windows\start_server.ps1
```

### 查看配置
```powershell
# 显示当前配置
python scripts\show_config.py

# 验证配置
python scripts\validate_config.py
```

### 文档查看
所有项目文档现在统一存放在 `docs/` 目录中，便于管理和查找。
