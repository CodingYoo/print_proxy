========================================
  Print Proxy Build Guide
========================================

Quick Build (Recommended)
--------------------------
Command: build.bat
Time: 2-3 minutes
Tests: Skipped (for speed)

Full Build (With Tests)
------------------------
Command: build_full.bat
Time: 3-5 minutes
Tests: Included

Manual Build Options
--------------------
# Quick build (skip tests)
powershell -ExecutionPolicy Bypass -File scripts\windows\build_exe.ps1 -SkipTests

# Full build (with tests)
powershell -ExecutionPolicy Bypass -File scripts\windows\build_exe.ps1

# Clean build (recreate environment)
powershell -ExecutionPolicy Bypass -File scripts\windows\build_exe.ps1 -Clean -SkipTests

Output
------
Location: dist\PrintProxy.exe
Size: ~51 MB

Run the EXE
-----------
dist\PrintProxy.exe

Access Service
--------------
http://localhost:8568/admin
http://localhost:8568/docs

Default Login
-------------
Username: admin
Password: admin123

Troubleshooting
---------------
1. Build fails?
   - Try: build.bat (uses -SkipTests)
   - Or: powershell -ExecutionPolicy Bypass -File scripts\windows\build_exe.ps1 -Clean -SkipTests

2. Port already in use?
   - Stop existing service
   - Or change port in .env file

3. Need help?
   - Check: scripts\windows\README.md
   - Check: QUICK_START.md
